# fotos/

Registro fotográfico das antenas avaliadas e dos ensaios realizados.

## antenas/

Uma foto por antena candidata, para documentação visual e referência rápida.

| Arquivo | Antena |
|---|---|
| `A0_foto.png` | Antena nativa do módulo ESP32 (integrada PCB) |
| `A1_foto.png` | Externa cilíndrica com cabo |
| `A2_foto.png` | Externa flexível longa (descartada) |
| `A3_foto.png` | Externa flexível média adesiva |
| `A4_foto.png` | Micro antena rígida SMD — finalista |
| `A5_foto.png` | Flexível FPC Molex T-Bar adesiva — finalista |

## Como Adicionar Novas Fotos

Nomeie os arquivos de forma descritiva seguindo o padrão:
- Fotos de setup/bancada: `ensaio_YYYY-MM-DD_descricao.jpg`
- Fotos de campo: `campo_YYYY-MM-DD_maquina_descricao.jpg`
