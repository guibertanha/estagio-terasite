# docs/

Procedimentos operacionais, checklists e guias para execução dos ensaios.

## Documentos

| Arquivo | Conteúdo |
|---|---|
| `checklist-dia1-esp32.md` | Checklist de configuração inicial do ESP32 para os ensaios. Verificações de hardware, upload de firmware e validação da comunicação serial. |
| `plano-rede-testes.md` | Plano da rede de testes: configuração do AP de referência, IPs, canais Wi-Fi e parâmetros de isolamento RF. |
| `procedimento-teste-minimo.md` | Procedimento mínimo viável para um ensaio de RSSI. Passo a passo para operador com o equipamento montado e pronto. |
| `guia-iperf.md` | Guia rápido de uso do iPerf3 para os ensaios de throughput em campo. Inclui instalação, comandos servidor/cliente e interpretação dos resultados. |
